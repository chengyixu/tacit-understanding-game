# 默契小游戏 - 部署说明

## 版本信息
当前版本: v2.2.0 - 矩阵相关性计算

## 部署后端服务器

使用提供的部署脚本：
```bash
chmod +x deploy.sh
./deploy.sh
```

该脚本会：
1. 将 server0405.py 复制到服务器
2. 重启 Python 服务
3. 验证服务运行状态

## 默契度计算说明
使用皮尔逊相关系数分析两位玩家的选择偏好矩阵：
- 构建 10×10 偏好矩阵
- 计算矩阵相关性
- 转换公式：默契度 = (相关系数 + 1) × 50
